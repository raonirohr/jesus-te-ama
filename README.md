# Jesus Te Ama

Ferramenta gratuita de evangelismo digital.

## Publicação
Envie todos os arquivos deste pacote para o repositório GitHub Pages.

## Estrutura
- `index.html`: página principal
- `css/style.css`: estilos
- `js/app.js`: interações
- `pages/`: páginas complementares
- `manifest.json` e `service-worker.js`: base PWA
